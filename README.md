MNN人脸检测轻量化模型部署，模型参数量203KB，适合大部分移动端设备运行。

### prepare

1. protobuf,opencv等第三方库编译
2. 编译MNN，参考github MNN文档。

### make

1. cd build
2. cmake ../
3. make

### execute

- cd build
- ./facedet
